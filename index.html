Luas Segitiga


Rumus luas segitiga yaitu:

L = 1/2 x a x t

Catatan:
L = Luas
a = Panjang alas
t = Tinggi

Keliling Segitiga


Rumus keliling segitiga yaitu:

K = S1 + S2 + S3

Catatan:
K = Keliling
S1 = Sisi 1
S2 = Sisi 2
S3 = Sisi 3

Luas Jajar Genjang


Rumus luas jajar genjang yaitu:

L = a x t

Catatan:
L = Luas
a = Alas
t = Tinggi

Hitung Luas Jajar Genjang










Keliling Jajar Genjang


Rumus keliling jajar genjang yaitu:

K = 2 x ( a + b )

Catatan:
K = Keliling
a = Alas
b = Sisi Miring
